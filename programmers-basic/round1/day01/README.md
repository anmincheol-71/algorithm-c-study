## 🔍 문제별 상세 기록

### 1. 문자열 출력하기
- **핵심 로직:** scanf()로 입력 받아 s1 배열에 담은 후 출력
- **사용한 개념:** 문자열 배열
- **회고:** 첫 시작인 만큼 가벼운 마음으로 풀었다. 무거운 마음으로 시작하면 오래가지 못할까봐

### 2. a와 b 출력하기
- **핵심 로직:** 두 정수를 입력 받아 각각 변수에 담고 출력 
- **사용한 개념:** 포맷팅
- **회고:** 줄 바꿈 문자 (\n)을 출력 시작에도 사용했더니 에러가 났다. 지우니깐 정상적으로 테스트 통과

### 3. 문자열 반복해서 출력하기
- **핵심 로직:** 입력받은 문자열과 정수를 사용하여 반복문 속에서 문자열 출력
- **사용한 개념:** 반복문
- **회고:** 정수 i 선언 후 반복문을 사용해서 코드가 길어진 것 같다는 생각 -> round2에선 코드 길이를 줄여봐야겠다

### 4. 대소문자 바꿔서 출력하기
- **핵심 로직:** 아스키코드를 참고하여서 대소문자 변경(+- 32)
- **사용한 개념:** 아스키코드, 조건문
- **회고:** 오타를 조심해야겠다. 122를 112로 적고 바로 알아차리지 못함

### 5. 특수문자 출력하기
- **핵심 로직:** 역슬래시 활용하여서 출력 -> 이스케이프 처리
- **사용한 개념:** 역슬래시 - 공부 후 -> 이스케이프 시퀀스, 문자열 리터럴 제어
- **회고:** printf 속 문자열이 너무 더럽다. 아스키 코드를 적는게 더 깔끔하겠다. / 개념을 그냥 역슬래시라고 적었는데 개념에 대해 공부를 해야겠다. -> 수정하였음
